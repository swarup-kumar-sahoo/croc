# Changelog

All notable changes to croc-ui will be documented here.

## [1.0.0] - 2025

### Added
- Initial release of croc-ui
- 30+ UI components: Card, Navbar, Modal, Tabs, Accordion, Table, Badge, Alert, Progress, Spinner, Avatar, Tooltip, Pagination, Breadcrumb, Sidebar, and more
- Full form components: Input, Button (8 variants, 3 sizes), Select, Textarea, Checkbox, Radio, FileInput, Range
- Layout system: Container, Row, Column, Section, Header, Footer, Div
- Typography: H1–H6, P, Code, Pre, Blockquote, A, Label, Span
- Media components: Img, Video, Audio, Icon, SVG
- Fluent `Style()` builder with 60+ CSS property methods
- Theme system with presets: `default`, `dark`, `ocean`, `forest`
- `Page` builder with Google Fonts, icon libraries, meta tags, custom CSS/JS
- `App` dev server with route registration, static file serving, browser auto-open
- Zero external dependencies (pure Python standard library)
- Python 3.8+ support
